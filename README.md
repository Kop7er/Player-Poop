# Player-Poop
Makes a player poop

## Permissions

* `playerpoop.ignore` -- Blocks player from pooping

## Configuration

``` json
{
  "Probability Of Pooping (1-100)": 50,
  "Only Poop If The Food Bar Is At Max": false
}
```
